4.2 Website Structure
A website may contain so many pages. A website can contain many web pages. Website structure
refers to how the pages of the website will be designed. Therefore, a structure on which all
information on a website can be presented is calledwebsite structure. There can be pages of
different types in a website structure.Among them, the most important are- Home Page, Main
Section Page, and Sub-section Page.
Home Page: The page that is shown first after entering a website is called a home page. The home
page is the platform to enter the website. Ideas about the context, target, and purpose of the
website are provided on the home page. Generally, there are multiple pages under the home page
and the list ofall pages in the form of an index is shown on the home page. It can be known from
the index the number of existing main sections. In terms of creating a home page, the noticeable
points are to keep nice and tidy by not keeping too much information and links should be provided
only for main section pages.

\formation. It is similar to a software menu or book index. From here it can be known that in the
ebsite users will get which information on which section. In terms of creating a main section
age, the noticeable points are not to keep similar category topics for the same information, to
ecp the links of sub-section pages including all information on specific subjects, to keep menu
nks to browse among the pages of the main section, etc.
ub-section Page: Information about specific topics is arranged on sub-section pages. The main
